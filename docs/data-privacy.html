Data Security and Privacy Statement

for Bitbucket SSH Support for Pull Request

tl;dr - This plugin does not store any data!

To make it more explicit, the plugin:

  • does not store personal data
  • does not communicate with anything outside the server
  • does not collect any usage data or statistics
  • all data is stored via regular API from hosting Atlassian product
  • any user does not get access to more data then without the plugin

    • We believe this is the best way to make sure no privacy issues can occur.
